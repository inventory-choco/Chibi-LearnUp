# Chibi Learn Up Website

Demo website for Chibi Learn Up, an online learning and tuition platform.

## Current Features

- Home page
- About section
- Class/program details
- Booking enquiry form
- WhatsApp enquiry connection
- Payment gateway preview section
- Mobile responsive design
- Contact links for Instagram and Facebook

## Future Features

- Online class booking system
- Student/parent login
- Admin panel
- Fee payment integration using Razorpay
- Email/SMS confirmation
- Class schedule management
